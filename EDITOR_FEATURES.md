# 图片编辑器功能说明

## 新增功能

### 1. 图形绘制工具

#### 矩形工具
- 拖拽绘制矩形
- 支持填充色和描边色
- 可通过拖拽角点调整大小

#### 圆形/椭圆工具
- 拖拽绘制圆形或椭圆
- 支持填充色和描边色
- 可通过拖拽角点调整大小

#### 直线工具
- 拖拽绘制直线
- **使用起止点控制**：选中后显示两个白色圆点，拖拽圆点调整直线的起点和终点
- 不会因缩放而变形，始终保持线条粗细

#### 箭头工具
- 拖拽绘制箭头
- **使用起止点控制**：选中后显示两个白色圆点，拖拽圆点调整箭头的起点和终点
- 箭头方向自动跟随终点位置
- 箭头大小根据线宽自动调整
- 不会因缩放而变形

### 2. 橡皮擦重构

**旧方式问题**：
- 橡皮擦画白色线条，会破坏底图像素

**新方式**：
- 点击对象即可删除
- **完全不影响底图**
- 只删除用户绘制的图形和画笔轨迹

### 3. 撤销/恢复修复

**旧方式问题**：
- 撤销时底图可能丢失

**新方式**：
- 底图永远不进入历史栈
- 只记录用户绘制的对象层
- 撤销/恢复时底图始终保持不变

### 4. 属性面板增强

#### 选中对象时
- 自动显示当前对象的属性
- 描边颜色：12色预设 + 自定义色盘
- 填充颜色：支持"无填充"选项（矩形、圆形）
- 线条粗细：1-50px 滑块调节
- 不透明度：10-100% 滑块调节

#### 实时修改
- 所有属性修改立即生效
- 每次修改自动保存到历史栈
- 支持撤销/恢复

### 5. 画笔功能优化

- 画笔绘制的路径是 Fabric.js 的 Path 对象
- 可以选中画笔轨迹
- 选中后可修改颜色和不透明度（线宽在绘制时确定）

## 使用方法

### 绘制图形
1. 点击左侧工具栏选择工具（矩形/圆形/直线/箭头）
2. 在画布上拖拽绘制
3. 松开鼠标确认

### 修改直线/箭头
1. 点击选择工具（Move 图标）
2. 点击直线或箭头选中
3. 拖拽两端的**白色圆点**调整起止点位置
4. 拖拽中间区域可整体移动

### 修改矩形/圆形
1. 选中对象
2. 拖拽角点调整大小
3. 拖拽中间区域移动
4. 在右侧面板修改填充色、描边色、线宽

### 删除对象
- 方式1：选中后按 Delete 或 Backspace 键
- 方式2：选中后点击右侧面板"删除选中"按钮
- 方式3：使用橡皮擦工具点击对象

### 撤销/恢复
- 撤销：Ctrl+Z（Mac: Cmd+Z）
- 重做：Ctrl+Shift+Z（Mac: Cmd+Shift+Z）
- 或点击左侧工具栏底部的撤销/重做按钮

## 技术实现

### 架构特点
- **数据驱动**：所有图形都是 Fabric.js 对象，存储完整属性
- **双层分离**：底图层（backgroundImage）+ 编辑层（objects）
- **对象模型**：支持选中、变换、属性修改
- **自定义控制器**：直线和箭头使用自定义的起止点控制器

### 关键技术点
1. **起止点控制器**：使用 Fabric.js 的 `Control` API 实现自定义控制点
2. **箭头几何更新**：拖拽端点时实时重新计算箭头角度和位置
3. **历史栈优化**：序列化时排除 backgroundImage，恢复时重新挂载
4. **闭包陷阱修复**：使用 `useRef` 让 Canvas 事件处理器读取最新状态

### 不变形原理
- 直线和箭头禁用了默认的缩放和旋转控制器
- 使用自定义的起止点控制器，直接修改端点坐标
- `strokeUniform: true` 确保线宽不随缩放变化
